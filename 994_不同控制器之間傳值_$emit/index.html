1. $scope.$emit 只能向父輩(包含爺輩) 傳遞 event 與 data

  2. $scope.$broadcast 只能向子代(包含孫代)傳遞 event 與 data

  3. $scope.$on 用於接收event與data

  4. $rootScope.$emit 任意處皆可傳/接

  5. $rootScope.$broadcast 任意處皆可傳/接

  6. $rootScope.$on 任意處皆可傳/接
